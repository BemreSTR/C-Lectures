1.Struct Defination

Tabii! Aşağıda class, object, member function, member variable, public, private, ve scope resolution operator gibi C++ kavramlarını içeren bir örnek kodu, detaylı açıklamalar ve yorumlarla birlikte bulabilirsiniz:

#include <iostream>
using namespace std;

// Bu bir class tanımıdır. Class, aynı özelliklere (attributes/member variables) ve davranışlara
// (functions/member functions) sahip bir grup nesnenin (objects) şablonudur.
class Araba {
private:
    // Bu bölümde, member variables (üye değişkenler) tanımlanır.
    // Private erişim belirleyicisi altında tanımlandıkları için dışarıdan doğrudan erişilemezler.
    string marka;
    int yil;

public:
    // Public erişim belirleyicisi, bu class'ın dışındaki kod tarafından erişilebilen
    // fonksiyonları ve değişkenleri belirtir.

    // Bu bir constructor fonksiyonudur. Constructor, nesne oluşturulduğunda (object)
    // otomatik olarak çalışan bir özel member function'dır.
    Araba(string m, int y) {
        marka = m;  // Member variable "marka" sınıfın içinden kullanılır.
        yil = y;    // Member variable "yil" sınıfın içinden kullanılır.
    }

    // Bu bir member function'dır. Class içindeki fonksiyonlar, class'ın davranışlarını tanımlar.
    // Public olduğu için dışarıdan erişilebilir.
    void arabaBilgileriniGoster() {
        cout << "Marka: " << marka << endl;  // Member variable 'marka' kullanılıyor.
        cout << "Yıl: " << yil << endl;      // Member variable 'yil' kullanılıyor.
    }

    // Bir başka member function. Bu fonksiyon member variable'ı değiştirmek için kullanılır.
    void setMarka(string yeniMarka) {
        marka = yeniMarka;
    }

    // Member function için scope resolution operator (::) kullanımı:
    // Bu operator, bir member function'ın bir sınıfa ait olduğunu belirtir.
    // Function burada tanımlanmış gibi dursa da, sınıfın dışında yazılabilir.
    int getYil() const;
};

// Bu kısımda, scope resolution operator (::) kullanıyoruz.
// "getYil" fonksiyonunun aslında "Araba" class'ına ait olduğunu belirtiriz.
int Araba::getYil() const {
    return yil;
}

int main() {
    // Bu bir object (nesne) oluşturma işlemidir.
    // Bir class'tan nesne oluşturmak, o sınıfın bir örneğini yaratmak anlamına gelir.
    Araba araba1("Toyota", 2020);  // araba1 isimli bir Araba nesnesi (object) oluşturuldu.

    // Public olan member function'lar, nesne üzerinden çağrılabilir.
    // Aşağıda araba1 nesnesi üzerinden member function çağırılıyor.
    araba1.arabaBilgileriniGoster();

    // Public olan başka bir member function çağrısı.
    araba1.setMarka("Honda");
    cout << "Marka değiştirildi: " << endl;
    araba1.arabaBilgileriniGoster();

    // Scope resolution operator kullanılarak tanımlanan getYil fonksiyonu da nesne üzerinden çağrılabilir.
    cout << "Arabanın üretim yılı: " << araba1.getYil() << endl;

    return 0;
}

Kısa Açıklamalar:

  • class: Sınıf, bir veri tipi gibi çalışır ve aynı özelliklere sahip nesnelerin şablonunu oluşturur. Bu örnekte Araba sınıfı tanımlanmıştır.
  • object: Bir sınıfın örneği (instance) olan nesnedir. araba1 nesnesi Araba sınıfının bir örneğidir.
  • member variable: Sınıfa ait değişkenlerdir. Bu örnekte marka ve yil, Araba sınıfının private olarak tanımlanmış member variable'larıdır. Üye değişkenler bir nesnenin özelliklerini tutar.
  • member function: Sınıfa ait fonksiyonlardır. Örneğin, arabaBilgileriniGoster() ve setMarka() fonksiyonları Araba sınıfının member function'larıdır. Bu fonksiyonlar sınıfın içinde tanımlandığı için o sınıfa ait nesneler üzerinde işlem yapabilir.
  • public: Erişim belirleyicisidir. public olarak işaretlenmiş fonksiyonlara ve verilere sınıf dışından erişilebilir. arabaBilgileriniGoster() ve setMarka() gibi fonksiyonlar public olarak tanımlanmıştır.
  • private: private erişim belirleyicisi altındaki değişkenlere ve fonksiyonlara yalnızca sınıf içinden erişilebilir. Dışarıdan doğrudan erişim mümkün değildir. Örneğin, marka ve yil private olarak tanımlanmış olup, yalnızca sınıfın içindeki fonksiyonlar tarafından kullanılabilir.
  • scope resolution operator (::): Bu operatör, bir fonksiyonun veya değişkenin hangi sınıfa ait olduğunu belirtmek için kullanılır. int Araba::getYil() ifadesinde, getYil() fonksiyonunun Araba sınıfına ait olduğunu gösterir.

Program Çıktısı:

Programın çalışması durumunda aşağıdaki gibi bir çıktı alırsınız:

Marka: Toyota
Yıl: 2020
Marka değiştirildi:
Marka: Honda
Yıl: 2020
Arabanın üretim yılı: 2020

Bu örnek ile C++'ta sınıfların, nesnelerin, üye fonksiyonların ve değişkenlerin nasıl kullanıldığını ve nasıl birbirleriyle ilişkili olduğunu öğrenmiş olursunuz.
